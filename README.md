# University Coursework — Mechanical Engineering (MSc)

A collection of computational coursework projects completed as part of an MSc in Mechanical Engineering. Each module folder contains the analysis notebooks, supporting scripts/data, and its own README describing the problem, methodology, and results.

## Modules

### [`KB7001`](./KB7001) — Computational Fluid Dynamics & Heat Transfer
Analytical modelling of a shell-and-tube oil-to-water heat exchanger, comparing plain and finned tube configurations using Reynolds/Nusselt number correlations.

**Tools:** Python (Jupyter Notebook), NumPy, Matplotlib, CSV/Excel data export.

### [`KB7043_Assignment1`](./KB7043_Assignment1) — Multidisciplinary Design & Optimisation
Optimisation of a passive quarter-car suspension model using Particle Swarm Optimisation (PSO) to minimise ride-acceleration variance, followed by a Monte Carlo sensitivity/robustness analysis.

**Tools:** Python (Jupyter Notebook), NumPy, SciPy, PySwarms, pyDOE, Matplotlib.

## Getting Started

To run the notebooks in this repository:

```bash
pip install numpy scipy matplotlib pandas pyswarms pyDOE
```

Then open the relevant `.ipynb` file in Jupyter Notebook, JupyterLab, or VS Code with the Jupyter extension.
